Next.js scaffold for consorcio2

This folder is an initial scaffold to migrate the existing Vite React app into Next.js.

How to run (from repository root):

1. cd next-app
2. npm install
3. npm run dev

Notes:
- This is a minimal starting point with dynamic route `pages/cartas/[id].tsx` example.
- To finish migration, move components from the root `components/` into this app (or import them with adjusted paths), update styles and Firebase imports.
